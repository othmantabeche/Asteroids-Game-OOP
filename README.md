# Asteroids Game

A classic Asteroids arcade game built with Python and Pygame.

## About

This project was built following the [Boot.dev Asteroids guided project](https://www.boot.dev/courses/build-asteroids-python) - a course focused on object-oriented programming principles with Python and Pygame.

## Requirements

- Python 3.13+
- Pygame

## Installation

```bash
pip install pygame
```

## How to Play

```bash
python main.py
```
